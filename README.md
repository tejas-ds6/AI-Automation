# 🤖 AI Job Matcher (Resume-Based Job Recommendation System)

## 🚀 Overview

This project is an AI-powered job recommendation system that:

* Extracts skills from a user's resume (PDF)
* Scrapes real-time job listings from Internshala
* Matches user skills with job requirements
* Ranks jobs based on match percentage

---

## 🛠️ Tech Stack

* Python
* BeautifulSoup (Web Scraping)
* Streamlit (Web App)
* PDFPlumber (Resume Parsing)
* Regex / NLP (Skill Extraction)

---

## 🔥 Features

* 📄 Resume upload (PDF)
* 🧠 Automatic skill extraction
* 🌐 Live job scraping
* 🎯 Skill-based job matching
* 📊 Match score calculation

```bash

```

## 📂 Project Structure

* `job_scraper.py` → Fetch jobs
* `Resume_reader.py` → Resume parsing & skill extraction
* `matcher.py` → Matching logic
* `app.py` → Streamlit UI
